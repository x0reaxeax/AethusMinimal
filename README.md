# AethusMinimal
Building/Placement restrictions remover for AETHUS (v1.029)

## Usage
Inject `AethusMinimal.dll` into `StrikeTheEarth-Win64-Shipping.exe`.  

Injector can be found in [Releases](https://github.com/x0reaxeax/AethusMinimal/releases).
